# productsPolymorphism
Um programa simples de criação de produtos em Java,que trabalha tópicos como, sobreposição, criação de classes, encapsulamento, uso de métodos getters e setters, e o polimorfismo. *Este código não realiza/considera tratamento de exceções.  Projeto feito de forma manual e própria. Baseado no curso do Prof.Dr. Nelio Alves.
